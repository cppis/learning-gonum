# Learning Gonum  

[Gonum](https://github.com/gonum/gonum)은 golang 으로 작성한 수에 대한 라이브러리 입니다.  
Gonum 을 사용한 확률 계산을 소개합니다.  

<br/><br/><br/>

## Apps  

### binomial  

[이항 분포(Binomial distribution)](https://en.wikipedia.org/wiki/Binomial_distribution)를 계산하는 예제입니다.
